Rank Rx

Steps to recreate exepriment.
1. Setup environment  
`conda create -n rankenv python=3.10`
2. Activate environment  
`conda activate rankevn`
3. Check the Anaconda Navigator > Environments, to see if the created env is activated and if not, manually select it.
5. Install Tensorflow Ranking  
`pip install -q tensorflow-ranking`
6. Check Tensorflow Ranking version, it should be Version: 0.5.5  
`pip show tensorflow-ranking`
7. Check Tensorflow version, it should be Version: 2.15.1  
`pip show tensorflow`
8. Check if Jupyter Notebook is installed on the created environment
9. Launch Jupyter Notebook  
`jupyter notebook`
